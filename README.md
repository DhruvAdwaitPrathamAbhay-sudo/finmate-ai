# 💰 FinMate AI

AI-powered finance assistant designed for students to manage expenses, track budgets, and get smart financial insights.

---

## 🚀 Features

- 📊 Expense Tracker (Add/Edit/Delete)
- 🤖 AI Financial Assistant (Smart insights)
- 📈 Spending Predictions
- 📅 Weekly & Monthly Analytics
- 🎯 Goal-Based Savings
- ⚠️ Overspending Alerts
- 🔐 Secure Authentication

---

## 🧠 How It Works

FinMate AI analyzes user spending patterns and provides:
- Personalized financial advice
- Risk level detection
- Budget optimization suggestions

---

## 🛠️ Tech Stack

- Frontend: React
- Backend: (AI Builder / Local Logic)
- Charts: Chart.js
- AI: Simulated intelligent system
